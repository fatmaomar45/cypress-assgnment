## Reflection

This Cypress testing project helped me understand the fundamentals of automated testing and how it is used in real-world software development. At the beginning, I learned how to structure test cases using describe and it blocks, which helped me organize my tests in a clear and readable way.

I also learned how to interact with different web elements such as buttons, input fields, checkboxes, dropdowns, and tables. This improved my understanding of how web applications behave and how users interact with them.

Another important skill I gained was writing assertions. I learned how to verify that elements exist, are visible, and contain the correct text. This is important in ensuring that applications work as expected.

I also practiced handling waiting mechanisms, which helped me understand how to deal with dynamic content that loads at different times. Working with forms and tables improved my ability to automate more complex user interactions.

Overall, this project gave me practical experience in end-to-end testing using Cypress. I now feel more confident in writing automated tests and understanding how QA engineers ensure software quality before release.
